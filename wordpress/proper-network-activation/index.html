Proper Network Activation: Version 1.0

July 09, 2010

This plugin came as a solution to the recently discovered problems with network activation on MultiSite installs.

The plugin does two things at the moment:

  • when doing a network de/activation, it triggers the de/activation hook on all sites in the network
  • when creating a new site, it triggers the activation hook for all active network plugins on that site
